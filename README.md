# 🚀 Desafios de Python

Este repositório contém meus estudos práticos em Python, indo do básico até lógica aplicada em problemas reais.

## 📚 Sobre

Sou estudante de Análise e Desenvolvimento de Sistemas e estou em transição de carreira para a área de TI.  
Este projeto faz parte do meu processo de aprendizado focado em lógica de programação e desenvolvimento.

## 🧠 Conteúdos praticados

- Entrada e saída de dados
- Estruturas condicionais (if/else)
- Loops (for e while)
- Cálculos matemáticos
- Lógica de negócios (simulação de corridas tipo Uber)
- Acumuladores e métricas (R$/km, total do dia)
- Estrutura de sistemas com menu interativo
